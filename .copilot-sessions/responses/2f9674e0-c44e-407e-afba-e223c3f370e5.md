## What I built

A **Live Grid** dashboard (new default top-bar tab; the old single-symbol view moved behind an **Analyse** tab), built enterprise-style around a pluggable data layer.

**Pluggable market-data layer** ([web/src/lib/marketData/](web/src/lib/marketData))
- [types.ts](web/src/lib/marketData/types.ts) — one `MarketDataSource` interface (`fetchCandles` + `subscribe`) any broker implements.
- [hyperliquid.ts](web/src/lib/marketData/hyperliquid.ts) — live crypto: a singleton **WebSocket multiplexer** (ref-counted `candle` subscriptions, heartbeat, exponential-backoff reconnect) + REST `candleSnapshot` history.
- [yfinance.ts](web/src/lib/marketData/yfinance.ts) — equities (US/SG/India) via the existing proxy/native path, "live" via change-gated polling.
- [registry.ts](web/src/lib/marketData/registry.ts) — `registerSource()` so Alpaca/Binance/Zerodha/Polygon drop in with zero UI changes.

**Dashboard UI** ([web/src/components/dashboard/](web/src/components/dashboard))
- `LiveDashboard` — the **Number of charts** selector (1/2/4/6/8, max 8).
- `ChartGrid` — responsive CSS grid (1=full, 2=side-by-side, 4=2×2, 6=3×2, 8=4×2; collapses on small screens).
- `LivePane` — independent **source/symbol/timeframe** dropdowns + data lifecycle.
- `LiveChart` — Lightweight Charts candlestick with imperative streaming updates.
- `TickerBar` — colour-coded bar that **flashes green/red on every price change**.

**State & persistence** — [useDashboardStore.ts](web/src/store/useDashboardStore.ts) persists chart count *and* every pane's config to `localStorage` (validated on load); [gridLayout.ts](web/src/lib/gridLayout.ts) holds the layout math.

**Proxy** ([proxy/src/worker.ts](proxy/src/worker.ts)) — added `interval`/`range` params for intraday equities and broadened the symbol regex so `D05.SI` (Singapore) and `RELIANCE.NS` (India) tickers work.

## Run it
```bash
pnpm --dir web dev      # crypto works immediately (Hyperliquid, no setup)
pnpm --dir proxy dev    # optional: equities in the browser (Yahoo, no API key)
```

Tests: `gridLayout`, `hyperliquid` (fake-WS multiplexer), `registry`, `useDashboardStore`, and `TickerBar` added (docs + `docs/TESTING.md` updated).

Caveat: equities stream via polling (no Yahoo WebSocket); crypto is true WebSocket push.
